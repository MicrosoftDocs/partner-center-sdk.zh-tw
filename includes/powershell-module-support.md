合作夥伴通常會使用[合作夥伴中心 PowerShell 模組](https://github.com/microsoft/partner-center-powershell/)來管理其合作夥伴中心資源。 這是由合作夥伴社群維護的開放原始碼專案。 由於此模組是由合作夥伴社區進行維護，因此 Microsoft 未正式支援該模組。 如果您遇到問題，可以[從社群取得協助](https://stackoverflow.com/questions/tagged/partner+center)或[在 GitHub 上提出問題](https://github.com/microsoft/partner-center-powershell/issues)。
